# Mobility-analytics-BEMM457

An Infogram app for analyzing 2023 London Cycle Hire Datasets

Jayant Chawla

Please visit webpage: https://infogram.com/jayant-1hmr6g8jokk7o2n

Features:

Data Collection: GetData.py is used to fetch and clean data

Processing: Dataprocessing.py is used to process and analyse data ready for the visualizations

Visualization: Matplotlib-powered charts embedded in a Infogram UI.

This project is made for BEMM457 - Topics in Business Analytics
